## 🚀 Project Overview

HedgeFundAI is an AI-powered quantitative investment research platform that combines:

- 📊 Technical Analysis
- 🧠 Large Language Models (Gemini & Groq)
- 📰 Market Sentiment Analysis
- 💹 Stock & Crypto Market Data
- 🤖 Intelligent Investment Recommendations

The goal is to build a hedge-fund-style research assistant capable of analyzing financial markets using AI, quantitative indicators, and real-time market information.

# ⚙️ Installation

Clone Repository

```bash
git clone https://github.com/Kanishk164337/HedgeFundAI.git
```

Go inside

```bash
cd HedgeFundAI
```

Create Virtual Environment

```bash
python -m venv .venv
```

Activate

Windows

```bash
.venv\Scripts\activate
```

Install Dependencies

```bash
pip install -r requirements.txt
```

---

# 🔑 Environment Variables

Create a `.env` file.

```env
GROQ_API_KEY=
GOOGLE_API_KEY=
ALPHA_VANTAGE_API_KEY=
FINNHUB_API_KEY=
NEWS_API_KEY=
DEFAULT_MODEL=groq
```
